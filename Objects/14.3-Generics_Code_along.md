## ðŸ§© Step 1: Create a Generic Class `Storage<T>`

```java
import java.util.ArrayList;
import java.util.List;

public class Storage<T> {
    private List<T> items;

    public Storage() {
        items = new ArrayList<>();
    }

    // Add item
    public void addItem(T item) {
        items.add(item);
    }

    // Remove item
    public boolean removeItem(T item) {
        return items.remove(item);
    }

    // Get item by index
    public T getItem(int index) {
        if (index >= 0 && index < items.size()) {
            return items.get(index);
        }
        return null;
    }

    // Get all items
    public List<T> getAllItems() {
        return items;
    }

    // Display items
    public void displayItems() {
        System.out.println(items);
    }
}
```

* Generic class with `<T>`
* Type-safe collection inside a generic class (`List<T>`)
* Basic operations: add, remove, retrieve

## ðŸ§© Step 2: Create a Generic Method

A **generic method** to swap two elements in an array:

```java
public class GenericUtils {
    public static <T> void swap(T[] array, int i, int j) {
        if (i >= 0 && i < array.length && j >= 0 && j < array.length) {
            T temp = array[i];
            array[i] = array[j];
            array[j] = temp;
        }
    }
}
```

âœ… **Concepts reinforced:**

* Generic method `<T>`
* Type-safe manipulation of arrays

## ðŸ§© Step 3: Test with Different Types

```java
public class GenericsLab {
    public static void main(String[] args) {
        // 1. Storage for Strings
        Storage<String> stringStorage = new Storage<>();
        stringStorage.addItem("Apple");
        stringStorage.addItem("Banana");
        stringStorage.addItem("Cherry");
        stringStorage.displayItems(); // [Apple, Banana, Cherry]

        stringStorage.removeItem("Banana");
        stringStorage.displayItems(); // [Apple, Cherry]

        System.out.println("Item at index 1: " + stringStorage.getItem(1)); // Cherry

        // 2. Storage for Integers
        Storage<Integer> intStorage = new Storage<>();
        intStorage.addItem(10);
        intStorage.addItem(20);
        intStorage.addItem(30);
        intStorage.displayItems(); // [10, 20, 30]

        // 3. Using generic swap method
        String[] fruits = {"Apple", "Banana", "Cherry"};
        System.out.println("Before swap: " + java.util.Arrays.toString(fruits));
        GenericUtils.swap(fruits, 0, 2);
        System.out.println("After swap: " + java.util.Arrays.toString(fruits));
    }
}
```

**Expected Output:**

```
[Apple, Banana, Cherry]
[Apple, Cherry]
Item at index 1: Cherry
[10, 20, 30]
Before swap: [Apple, Banana, Cherry]
After swap: [Cherry, Banana, Apple]
```
